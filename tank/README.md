# 坦克大战（Tank Battle）Python 小游戏

本项目目标：用 **Python + Pygame** 实作一个可游玩的「坦克大战」小游戏（单机、俯视角），包含关卡、敌方 AI、碰撞与简易 UI。

新手学习用：请看 `docs/逐行讲解.md`（按文件逐行解释代码）。

---

## 需求文档（Requirements）

### 产品定位
- **类型**：2D 俯视角动作射击（坦克）
- **平台**：Windows 10（可延伸至 macOS/Linux）
- **单机模式**：单人游玩（玩家 1 台坦克 + 多台敌方坦克）
- **目标体验**：操作直觉、打击感清楚、可在 3–5 分钟内玩完一关

### 核心玩法
- **玩家坦克**
  - 方向：上/下/左/右移动
  - 射击：发射子弹（可限制射速/同屏子弹数）
  - 生命/护甲：被击中扣血，血量归零则失败
- **双人对战（PVP）**
  - 2 名玩家在同屏地图内对战
  - 子弹只会伤害对方（不会伤到自己）
  - 任意一方血量归零则本局结束
- **敌方坦克**
  - 巡逻/追击（简易 AI）
  - 具备射击能力
  - 敌方全部消灭或达成条件即过关
- **地图与障碍物**
  - 墙体：分为可破坏/不可破坏两种（至少先做不可破坏）
  - 边界：不可穿越
  - 出生点：玩家与敌方生成位置固定或依关卡配置
- **子弹与碰撞**
  - 子弹与墙/坦克碰撞判定
  - 子弹击中墙：墙可破坏则扣耐久；不可破坏则子弹消失
  - 子弹击中坦克：扣血并产生命中特效（简易粒子/闪烁也可）

### 关卡与胜负条件
- **关卡数**：至少 1 关（设计成可扩充为多关）
- **胜利条件（PVP）**：击败对手（对手血量归零）
- **失败条件（PVP）**：自己血量归零
- **关卡结束流程**：显示结算画面（胜/败、用时、击杀数）

### 操作与 UI
- **键盘操作（预设）**
  - P1 移动：W/A/S/D
  - P1 射击：Space
  - P2 移动：方向键
  - P2 射击：右 Ctrl（RCTRL）
  - 暂停：Esc
  - 重新开始：R（结算画面）
- **UI（最小可用）**
  - 左上角：P1 血量
  - 右上角：P2 血量
  - 暂停画面：继续/返回主菜单
  - 主菜单：开始游戏/退出

### 音效与美术（可逐步加）
- **美术**：可先用色块/简单 sprite（方块坦克 + 方块子弹）
- **音效**：射击/爆炸/命中（可选）

### 非功能需求（Non-functional）
- **性能**：在一般笔电上稳定 60 FPS（或固定 tick 60）
- **可维护性**：模块化（scene、entity、physics、ai、assets）
- **可扩充性**：关卡数据用 JSON/TMX/简易文字图（先用 JSON）
- **可复现性**：固定随机种子（便于除错，可选）

### 不在第一版范围（Out of scope for MVP）
- 联机对战、多人同屏
- 复杂物理（反弹、穿透等）
- 皮肤/商店/存档系统（可留做后续）

## 技术选型
- **语言**：Python 3.11+（建议）
- **引擎/框架**：`pygame`（渲染、输入、音效）
- **数据格式**：JSON（关卡配置）
- **工具**：`venv` + `pip`

## 开发计划（Development Plan）

### 里程碑 0：环境与骨架（0.5 天）
- 建立 `src/main.py`，完成 pygame 窗口、主循环、FPS 控制
- 建立基本的 Scene 管理（Menu / Gameplay / GameOver）
- 规划文件夹结构与基本设置（窗口大小、颜色、速度常数）

### 里程碑 1：玩家可移动与射击（0.5–1 天）
- 玩家坦克：位置、速度、方向、渲染（先用矩形）
- 射击：子弹生成、移动、生命周期（出界消失）
- UI：显示血量/剩余敌人（先用 pygame font）

### 里程碑 2：地图与碰撞（1 天）
- 墙体（不可穿越）
- 碰撞系统：坦克 vs 墙、子弹 vs 墙、子弹 vs 坦克
- 基础命中反馈（闪烁/简易特效）

### 里程碑 3：敌人 AI（1 天）
- 敌人生成与管理
- AI（MVP）：随机巡逻 + 看到玩家（距离/射线简化）就朝向玩家
- 敌人射击：射速限制 + 子弹同样走碰撞系统

### 里程碑 4：关卡数据化与流程（0.5–1 天）
- `levels/level_01.json`：墙、敌人出生点、玩家出生点
- 过关/失败判定与结算画面
- 重新开始/回主菜单

### 里程碑 5：打磨与扩充（可选）
- 可破坏墙（耐久度）
- 强化道具（加血、加速、射速提升）
- 音效、图片资产、简单动画
- 更自然的敌人 AI（避障、短路径、行为树/状态机）

  - 玩家移动、射击是否稳定
